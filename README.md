<h2>DC Carousel &ndash; Free PrestaShop Image Carousel Module</h2>
<p><strong>DC Carousel (Design Cart Carousel)</strong> is a free PrestaShop module that allows you to easily add a responsive image carousel to the homepage of your store.</p>
<p>The module supports multilingual slides, customizable appearance, optional intro content, and a lightbox gallery for displaying larger images with descriptions and links.</p>
<p>It is designed to be lightweight, flexible, and easy to manage directly from the PrestaShop admin panel.</p>
<hr/>
<h2>Key Features</h2>
<h3>Responsive Image Carousel</h3>
<p>Display a modern and responsive carousel on the homepage of your store.<br/> The carousel adapts to different screen sizes and allows separate configuration for desktop, tablet, and mobile devices.</p>
<h3>Multilingual Slides</h3>
<p>Each slide supports multiple languages with:</p>
<ul>
<li>
<p>title</p>
</li>
<li>
<p>description</p>
</li>
<li>
<p>optional link</p>
</li>
</ul>
<p>This allows you to present localized content depending on the store language.</p>
<h3>Lightbox Image Gallery</h3>
<p>Clicking a slide opens a full image preview using <strong>GLightbox</strong>, including:</p>
<ul>
<li>
<p>slide title</p>
</li>
<li>
<p>slide description</p>
</li>
<li>
<p>optional link (e.g. "Go to product")</p>
</li>
</ul>
<p>This creates a smooth gallery experience for users.</p>
<h3>Drag &amp; Drop Slide Ordering</h3>
<p>Slides can be reordered directly in the admin panel using drag &amp; drop, allowing quick control over the carousel content.</p>
<h3>Built-in File Manager</h3>
<p>The module includes a lightweight file manager that allows:</p>
<ul>
<li>
<p>browsing image folders</p>
</li>
<li>
<p>uploading new images</p>
</li>
<li>
<p>selecting images directly inside the admin panel</p>
</li>
</ul>
<h3>Fully Customizable Appearance</h3>
<p>The module provides appearance settings for:</p>
<ul>
<li>
<p>intro section (title and description)</p>
</li>
<li>
<p>carousel behavior</p>
</li>
<li>
<p>navigation buttons</p>
</li>
<li>
<p>image borders</p>
</li>
<li>
<p>aspect ratio</p>
</li>
<li>
<p>background color</p>
</li>
</ul>
<p>This allows store owners to adapt the carousel to their design without editing code.</p>
<hr/>
<h2>Technical Overview</h2>
<p>The module integrates with PrestaShop using standard hooks:</p>
<ul>
<li>
<p><strong>displayHome</strong> &ndash; renders the carousel on the homepage</p>
</li>
<li>
<p><strong>displayHeader</strong> &ndash; loads required CSS and JavaScript assets</p>
</li>
</ul>
<p>Frontend technologies used:</p>
<ul>
<li>
<p><strong>Owl Carousel</strong> &ndash; carousel functionality</p>
</li>
<li>
<p><strong>GLightbox</strong> &ndash; image lightbox</p>
</li>
<li>
<p><strong>Smarty templates</strong> &ndash; rendering frontend content</p>
</li>
</ul>
<p>The module uses database tables for slides and multilingual content, and configuration values stored in PrestaShop <code>Configuration</code>.</p>
